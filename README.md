# Pizza Sales Data Analysis

## Project Overview
This project analyzes pizza sales data to uncover key business insights related to revenue trends, customer ordering behavior, and product performance. Using SQL for data exploration and data visualization for reporting, the project transforms raw transactional data into meaningful insights that support better business decision-making.

## Business Problem
A pizza business wants to understand its sales performance and customer demand patterns. The goal of this analysis is to answer important business questions such as:
- When do customers place the most orders?
- Which pizza categories generate the highest sales?
- Which pizza sizes contribute the most to revenue?
- Which pizzas are the best and worst performers?

Understanding these patterns helps businesses optimize product offerings, improve sales strategies, and increase revenue.

## Dataset
The dataset contains transactional pizza sales data including:
- Order ID
- Order Date
- Order Time
- Pizza Name
- Pizza Category
- Pizza Size
- Quantity Ordered
- Unit Price
- Total Price

## Tools & Technologies Used
- SQL – Data querying and analysis
- Data Visualization (Power BI / Excel) – Dashboard creation
- GitHub – Project documentation and version control

## Key Performance Indicators (KPIs)
The analysis focuses on the following metrics:
- Total Revenue
- Average Order Value
- Total Pizzas Sold
- Total Orders
- Average Pizzas per Order

## Key Insights
- Orders are highest on weekends, particularly Friday and Saturday evenings.
- Peak ordering times occur between 12 PM – 1 PM and 4 PM – 8 PM.
- The Classic pizza category contributes the highest share of total sales.
- Large-sized pizzas generate the highest revenue among all pizza sizes.
- The top-selling pizzas include Classic Deluxe Pizza, Barbecue Chicken Pizza, and Hawaiian Pizza.
- The lowest-selling pizzas include Brie Carre Pizza, Mediterranean Pizza, and Calabrese Pizza.

## Dashboard
The dashboard provides a visual overview of key sales metrics, order trends, category performance, and product insights to better understand business performance.
